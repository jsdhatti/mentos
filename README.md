# MENTOS #

### Tech Stack? ###
1. NodeJS with HapiJS
2. [Aurelia](http://aurelia.io)
3. Mongodb

### Prerequisite? ###
1. Node >= 4.0.0
2. Grunt-cli (for backend)
3. Mocha
4. Mongodb
5. JSPM
6. Gulp (for frontend)

### Start contributing? ###
1. clone
2. npm install
3. cd front-end && npm install
4. cd front-end && jspm install -y

### Run? ###
1. npm start [option: dev | prod ---- (dev is default)]

### Test? ###
1. npm test
